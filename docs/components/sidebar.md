---
title: Sidebar
category: Components
---

A sidebar is used for adding additional navigation to the left side of a page.

<div class="sidebar">
  <div class="sidebar__title">
    Sidebar links
  </div>
  <ul class="sidebar__nav">
    <li>
      <a href="#">First link</a>
    </li>
    <li>
      <a href="#" class="link--active">Active link</a>
    </li>
  </ul>
</div>
