---
title: Lists
category: Base
---

Lists (`<ol />` and `<ul />`) have no styling by default:

<ul>
  <li>
    Item One
  </li>
  <li>
    Item Two
  </li>
  <li>
    Item Three
  </li>
</ul>

```html
<ul>
  <li>
    Item One
  </li>
  <li>
    Item Two
  </li>
  <li>
    Item Three
  </li>
</ul>
```

Add bullets to a list with the `list--bullet` class:

<ul class="list--bullet">
  <li>
    Item One
  </li>
  <li>
    Item Two
  </li>
  <li>
    Item Three
  </li>
</ul>

```html
<ul class="list--bullet">
  <li>
    Item One
  </li>
  <li>
    Item Two
  </li>
  <li>
    Item Three
  </li>
</ul>
```
