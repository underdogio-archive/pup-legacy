---
title: Tables
category: Base
---

Use a table whenever you want to present tabular data.

This should go without saying, but tables should never be used for layout.
Unless you're writing markup for an email.

<table>
  <thead>
    <tr>
      <th>Location</th>
      <th>Plan</th>
      <th>Created</th>
      <th>Status</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>New York</td>
      <td>Free</td>
      <td>Jul 6, 2016</td>
      <td>Active</td>
    </tr>
    <tr>
      <td>Los Angeles (coming soon)</td>
      <td>Free</td>
      <td>Aug 3, 2016</td>
      <td>Active</td>
    </tr>
    <tr>
      <td>Remote</td>
      <td>Free</td>
      <td>Jul 6, 2016</td>
      <td>Active</td>
    </tr>
    <tr>
      <td>San Francisco</td>
      <td>Free</td>
      <td>Jul 6, 2016</td>
      <td>Active</td>
    </tr>
  </tbody>
</table>
